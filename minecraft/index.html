Join Steps

Quick path to get online.

  1. Open Minecraft Java Edition and add a new multiplayer server.
  2. Use answers-advertising.gl.joinmc.link:25565 as address.
  3. If offline, request boot in Discord using !mcup.
  4. Wait for the bot ready message, then join.
